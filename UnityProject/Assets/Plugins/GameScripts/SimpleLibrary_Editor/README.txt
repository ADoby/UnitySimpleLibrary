This is SimpleLibrary Editor
an extension for Unity3D Editor.

It adds:
=> Simple Editor Shortcuts (Mostly included in newer Unity versions)
SHIFT + ALT + A => Toggle active state of selected gameObjects
SHIFT + ALT + N => Create empty child gameObject
SHIFT + ALT + W => Wrap selected objects in one empty parent gameObject

=> Extended Transform Inspector
Reset (set to zero or 1) position, rotation and scale with or without editing child transforms
Round position, rotation and scale

=> Simple Pivot Editor
Edit pivot of selected gameObject (requires mesh) by moving the objects move handle
Center pivot of selected object
Shotcurt to open: SHIFT + ALT + P
